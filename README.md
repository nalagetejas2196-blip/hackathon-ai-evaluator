# AI Hackathon Evaluation System

This project is a simple prototype that allows users to submit problem statements and solutions, and get AI-based evaluation scores.

## Features
- Submission form for problem statement and solution
- Basic evaluation output (simulated scores and feedback)

## Tech Stack
- HTML, JavaScript

## How to Run
1. Open `index.html` in any web browser
2. Fill the Problem Statement and Solution fields
3. Click "Evaluate" to see scores and feedback
